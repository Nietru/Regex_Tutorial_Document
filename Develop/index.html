My Regex (AKA Regular Expressions) - Tutorial Notes

What is a meta character?

There will be subtle differences in what the meta-characters ARE depending on the programming language.
Link for meta-character tutorial: https://www.youtube.com/watch?v=YTocEnDsMNw
Meta-characters have 3 main sections:
  1. Single Chars: (sidenote: if you were to do a capital \D, it would be anything that ISNT a digit. Same with \W and \S) It represents the opposite.
    \d -> Indicates any digit, one character 0-9.
    \w -> Indicates any word, any character that you might find as part of a word, letter or number: A-Z a-z 0-9.
    \s -> Whitespace
    . -> ANY character what-so-ever!

  2. Quantifiers: Meta-characters that will match one or more characters in a row. We can find all 5 letter words like so: \s\w{5}\s
    * -> 0 or more.
    + -> 1 or more.
    ? -> 0 or 1.
    {min, max}
    {n}

  3. Position: Matches the position of a character IN the string itself:
    (sidenote: ) ^ -> "Hat" or "Carrot" Beginning.
    $ -> End.
    \b -> Word boundary.

  4. Character Classes: Matching:
    Link to tutorial: https://www.youtube.com/watch?v=EfJU0Y9WAZ4
    [ abc ] A bracket is not a meta character. Match any one the characters inside of the brackets.
    [ -. ] (sidenote: a dot inside of a character class is no longer a meta character.)

  5. Alternation:
    \. -> means literal dot.
